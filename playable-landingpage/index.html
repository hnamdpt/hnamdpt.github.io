What is a playable ads?

A playable ad is a mini HTML5 game experience, that is launched as an interactive ad.

Towards the end of the experience, the user is encouraged to install the full version of the app.

HTML5 playable ads are the next evolution after banner and video ads. They offer far better engagement rates and value for money.
